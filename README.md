# FFT benchmark
A benchmark for comparison of FFT algorithms performance.

Currently supports Intel IPP, [KFR](https://github.com/kfrlib/kfr), FFTW and KissFFT.

Requires:
* C++11 compiler (any recent version of GCC or Clang)
* MinGW on Windows
* CMake 2.8 or newer
* AVX-capable cpu
* Python 2.7 or newer
  * matplotlib module
  * numpy module
